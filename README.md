# HUB-PODIUM MVP

Produto inicial: Análise Inteligente de Atleta por vídeo + relatório comparativo + card estilo FIFA.

## Estrutura

- `mobile-app/` App Expo React Native
- `backend/` API Python Flask
- `landing-page/` Página comercial inicial
- `database/` Schema PostgreSQL/Supabase
- `docs/` Documentação executiva

## Fluxo do produto

1. Cliente acessa landing page
2. Compra análise de R$49
3. Cadastra atleta
4. Envia vídeo de treino ou jogo
5. Backend roda análise inicial
6. Sistema gera relatório e card do atleta
